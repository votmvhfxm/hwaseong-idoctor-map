# Clinics Cache

`clinics-cache.json` is generated by:

```powershell
node scripts\build-clinics-cache.js
```

The script writes the file only when the HIRA result passes the quality gate:

- at least 50 clinics
- at least 10 clinics with hours
- at least 50 verified pediatric clinics
- zero detail API failures

`api/clinics.js` serves this static cache first. If the file is missing or fails the quality gate, the API falls back to live HIRA collection and keeps bad responses as `no-store`.
